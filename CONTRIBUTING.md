# Contributing to ITR Filing Skill

## How to Contribute

### Reporting Issues
- Open a GitHub Issue
- Describe the tax scenario that is wrong or missing
- Mention the Assessment Year affected

### Submitting Changes
1. Fork the repository
2. Create a branch: `git checkout -b fix/description`
3. Edit files in `skill/` folder
4. Test by uploading SKILL.md to Claude
5. Submit a Pull Request

### What to Contribute
- ✅ Tax rule corrections
- ✅ New salaried employee scenarios
- ✅ Portal navigation updates
- ✅ New AY updates (slabs, limits)
- ✅ Better examples and calculations
- ❌ Do not add advice promoting tax evasion

## Annual Update Checklist
- [ ] Update tax slabs (New + Old Regime)
- [ ] Update standard deduction amount
- [ ] Update rebate u/s 87A limits
- [ ] Update LTCG exemption limit
- [ ] Update 80D / 80C limits if changed
- [ ] Update AY references throughout
- [ ] Update CHANGELOG.md
- [ ] Create new GitHub Release
